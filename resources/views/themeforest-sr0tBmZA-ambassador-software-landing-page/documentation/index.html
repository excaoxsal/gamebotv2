Video Popups

We use Magnific Popup for video popups, all you have to do is to add your video url and a class

Example: 
<a href="your-video-url" class="btn btn-link js-video-modal-trigger">
  watch the video
</a>

js-video-modal-trigger is the JavaScript hook for video modals

Utilities

This template uses most of the bootstrap the bootstrap's components and Utilities classes.getbootstrap.com

Custom Utilities classes

Spacings
  • m-t = margin-top
  • m-t = margin-left
  • p-t = padding-top
  • p-b = padding-bottom

Spacing classes name comes with the space values like mt-0 for margin top at zero and mt-10 for margin-top at 10px

Heights

All height Utilities are relative to the viewport height like height-20, which has 20% of the viewport height.

It has 10,20,30,40,50,60,70,80,90,10
